命令 作用
git branch -a 查看所有分支

git fetch 更新索引

git reset –hard 6e52 回退到某个 ID
git add -u 将删除操作也添加到暂存区
git rm –cached filename 将暂存区的文件移出暂存区

checkout 就是撤销你的修改, 暂存区是会保留.
