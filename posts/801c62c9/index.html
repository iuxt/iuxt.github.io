system 用户

服务默认的执行用户，如果一个程序在 windows 上直接运行和通过服务来运行效果不一样，可以考虑下服务执行的用户是不是当前用户，比如我用 syncthing ，直接运行配置好了一会，使用 always up 来运行配置丢失了，原因是配置文件被写入当前用户，而用 always up 默认使用的是 system 用户，其中 system 用户的用户文件夹在：
