先登录 redis

```bash
redis-cli -h 192.168.1.1
```

输入密码

```bash
192.168.1.1:6379> auth ******
```

直接删除

```bash
192.168.1.1:6379> del xxxxx
```

如果报错：

```bash
192.168.1.1:6379> del xxxxx
(error) MOVED 5704 192.168.1.2:6379
192.168.1.1:6379>
```

退出，登录 192.168.1.2, 重复上面的操作。
