## 安装 Django

```bash
python -m venv venv
source venv/bin/activate
python -m pip install Django==4.2
```

## 安装 Django restframework

```bash
pip install djangorestframework
```

```python
INSTALLED_APPS = [
    ...
    'rest_framework',
]
```

## 创建项目

```bash
django-admin startproject 项目名 文件夹
```

## 创建 APP

```bash
python manage.py startapp account
```

## 创建一个用户

```bash
python manage.py createsuperuser
```
