## dig

### 安装

```vim
# Ubuntu
apt-get install dnsutils

# CentOS
yum install bind-utils
```

### 查询

```bash
# 查询域名，指定DNS地址
dig auth.vipthink.cn @114.114.114.114

# 检查a记录
dig a www.baidu.com

# 检查cname记录
dig cname www.baidu.com

# 反向解析
dig -x 10.0.0.2
```

## nslookup

### 检查域名 A 记录

```bat
nslookup baidu.com 114.114.114.114
nslookup -type=a qq.com 114.114.114.114
nslookup baidu.com
```

### 检查域名 NS 记录

```bat
nslookup
> set type=ns
> zahui.fan 114.114.114.114
```

### 检查域名 CNAME 记录

```bat
nslookup
> set type=cname
> zahui.fan 114.114.114.114
```

### 反向解析

```bash
nslookup 10.0.0.2

# 指定dns服务器反向解析ip：10.0.0.2
nslookup 10.0.0.2 127.0.0.1
```
